## Project Overview
CineVerse is a **console-based Movie Ticket Booking System** written in Python.  
It allows users to **view available showtimes, book tickets, and check ticket details** while demonstrating Object-Oriented Programming concepts such as classes, attributes, and methods.  

This project is ideal for **students practicing Python** and building **interactive terminal applications**.
